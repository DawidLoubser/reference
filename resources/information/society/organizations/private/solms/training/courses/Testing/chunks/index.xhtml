Professional Software Testing

It is widely accepted that testing should be incorporated from the start into the analysis, design and development process. Not only does solid testing build confidence, it also makes systems and organizations much more flexible and responsive and increases the productivity of the development processes as a whole, i.e. "When quality is persued, productivity follows".

This course provides an in depth understanding of the purpose, responsibilities and approaches to testing and provides practical foundation of implementing a professional software testing paradigm.

Target audience

The course targets

  • Quality engineers

  • Technical testers

  • User acceptance testers

  • Software developers

  • Business analysts

  • Project managers
